Vélib'Mania, la passion du vélo !

Bienvenus chers visiteurs! Vous trouverez dans le diaporama ci-dessous toutes les informations à savoir lors de votre navigation, bonne visite !

Les marqueurs numérotés indiquent le nombre de stations. Cliquez sur un marqueur les verts sont les stations ouvertes et les rouges celles fermées.

Voici les détails de la station. Vous pouvez cliquer sur réserver pour accéder à la signature électronique, fermer ce panneau ou choisir une nouvelle station.

Maintenant, vous devez signer puis cliquer sur valider pour finaliser la réservation.

Une fois la signature validée, votre réservation est condirmée, vous aurez alors 20 minutes pour récupérer votre Vélib ! Merci d'avoir utilisé Vélib'Mania!
